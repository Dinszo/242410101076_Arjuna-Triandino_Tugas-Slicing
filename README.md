# 242410101076_Arjuna-Triandino_Tugas-Slicing
Tugas Slicing Pweb - Long Black

#penjelasan
Project ini adalah website landing page untuk penjualan kopi dengan nama Long Black. Website ini dibuat pakai HTML, CSS, dan JavaScript (DOM) tanpa framework, jadi semuanya dibuat manual (plain CSS).
Tampilannya dibuat modern dan sudah responsive, jadi bisa dibuka dengan nyaman di HP, tablet, maupun laptop.
Di dalam website ini ada beberapa fitur seperti:
Navbar yang bisa dibuka di mobile (menu toggle)
Tampilan utama buat promosi kopi
Section keunggulan produk
Daftar menu kopi yang bisa difilter
Testimonial yang bisa berubah otomatis
Notifikasi saat klik tombol beli
Form untuk menghubungi piihak long black
